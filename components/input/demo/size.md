## zh-CN

我们为 `nz-input` 输入框定义了三种尺寸（大、默认、小），高度分别为 `40px`、`32px` 和 `24px`。

注意： 在表单里面，我们只使用大尺寸的输入框。

## en-US

There are three sizes of an Input box: `large` (40px)、`default` (32px) and `small` (24px).

Note: Inside of forms, only the large size is used.
