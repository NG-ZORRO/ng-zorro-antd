## zh-CN

您可以为消息内容创建自定义模板。
通过 `nzData` 选项，您可以传递一些可选的数据给此自定义模板。

## en-US

You can have a custom template for the message content.
You would have the possibility to pass some optional data to this custom template thanks to the `nzData` option
