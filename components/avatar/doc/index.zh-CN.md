用来代表用户或事物，支持图片、图标或字符展示。

## API

### nz-avatar

| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| `[nzIcon]` | 设置头像的图标类型，参考 `Icon` | string | - |
| `[nzShape]` | 指定头像的形状 | Enum{ 'circle', 'square' } | `circle` |
| `[nzSize]` | 设置头像的大小 | Enum{ 'large', 'small', 'default' } | `default` |
| `[nzSrc]` | 图片类头像的资源地址 | string | - |
| `[nzText]` | 文本类头像 | string | - |
