## zh-CN

列偏移。

使用 `nzOffset` 可以将列向右侧偏。例如，`nzOffset="4"` 将元素向右侧偏移了 4 个列（column）的宽度。

## en-US

`nzOffset` can set the column to the right side. For example, using `nzOffset = "4"` can set the element shifted to the right four columns width.
