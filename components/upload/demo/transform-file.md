## zh-CN

> 旧的 `nzTransformFile` 属性已被弃用.

使用 `nzBeforeUpload` 在请求之前转换文件，例如添加水印。

## en-US

> The old `nzTransformFile` input has been deprecated.

Use `nzBeforeUpload` to transform file before requesting it, for example to add a watermark.
