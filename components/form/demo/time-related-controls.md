## zh-CN

时间类组件的输入和输出类型均为 `Date` 类型，可以通过 [date-fns](https://date-fns.org/) 工具库进行进一步的处理。

## en-US

All the types of input and output in time-related components are `Date`. You can use [date-fns](https://date-fns.org/) to handle it.
