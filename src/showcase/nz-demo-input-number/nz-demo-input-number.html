数字输入框

点击按钮切换可用状态。

通过 nzFormatter 格式化数字，以展示具有具体含义的数据，往往需要配合 nzParser 一起使用。

三种大小的数字输入框，当 nzSize 分别为 large 和 small 时，输入框高度为 32px 和 22px ，默认高度为 28px

和原生的数字输入框一样，value 的精度由 nzStep 的小数位数决定。

API

属性如下

成员 说明 类型 默认值
ngModel 当前值，可双向绑定 Number
nzPlaceHolder placeholder String ''
nzMin 最小值 Number -Infinity
nzMax 最大值 Number Infinity
nzStep 每次改变步数，可以为小数 Number or String 1
nzDisabled 禁用 Boolean false
nzSize 输入框大小 String 无
nzBlur 失去焦点回调 EventEmitter 无
nzFocus 获取焦点回调 EventEmitter 无
nzFormatter 指定输入框展示值的格式 function(value: number | string): string -
nzParser 指定从 nzFormatter 里转换回数字的方式，和 nzFormatter 搭配使用 function( string): number -
