Breadcrumbs

Shows where the current page is in the system hierarchy and can be returned up.

When to use #

  • When your system has more than two levels of hierarchy;

  • When you need to tell the user "where are you"?

  • When you need to navigate up the feature.

  • The a tag in breadcrumb can be associated with RouterLinkActive
