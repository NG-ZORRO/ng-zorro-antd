Switch 开关

开关选择器。

何时使用

  • 需要表示开关状态/两种状态之间的切换时；

  • 和 checkbox的区别是，切换 switch 会直接触发状态改变，而 checkbox 一般用于状态标记，需要和提交操作配合。

最简单的用法。

API

nz-switch

参数 说明 类型 介绍 默认值
ngModel 指定当前是否选中 Boolean 双向绑定 false
nzSize 开关大小 String 'default'|'small' 'default'
nzDisabled 开关是否禁用 String 'default'|'small' 'default'
checked 选中时的内容 selector 标记符
unchecked 非选中时的内容 selector 标记符
