Checkbox 多选框

多选框。

何时使用 #

  • 在一组可选项中进行多项选择时；

  • 单独使用可以表示两种状态之间的切换，和 switch 类似。区别在于切换 switch 会直接触发状态改变，而checkbox一般用于状态标记，需要和提交操作配合。

API

[nz-checkbox]

[nz-checkbox]一般配合label一起使用，具体见demo

参数 说明 类型 双向绑定 默认值
ngModel 指定当前是否选中 Boolean 是 false
nzIndeterminate 设置 indeterminate 状态，只负责样式控制 Boolean 否 false

nz-checkbox-group

参数 说明 类型 双向绑定 默认值
ngModel 所有选项 array 是 []
nzDisabled 是否可选 Boolean 否 false
