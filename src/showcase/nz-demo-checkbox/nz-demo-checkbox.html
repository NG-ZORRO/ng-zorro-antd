Checkbox.

When to use #

  • When making multiple selections in a group of options;

  • Used alone to indicate the switch between two states, similar to switch . The difference is that switching switch triggers state changes directly, whereas checkbox is generally used for stateful markup and needs to be associated with commit operations.
