Notificatio

Global display notification reminder information.

When to use #

A notification reminder appears in the top right corner of the system. Often used in the following situations:

  • More complicated notification content.

  • With interactive notifications, give the user the next action point.

  • The system is actively pushing.

How to use

Like Message, if you want to modify the default message configuration, you can modify the value of provider NZ_NOTIFICATION_CONFIG

(For example, adding {{'{'}} in your module's provider provides: NZ_NOTIFICATION_CONFIG, useValue: {{'{'}} nzDuration: 3000 {{'}'}} {{'}'}} , NZ_NOTIFICATION_CONFIG can be imported from ng-zorro-antd

The default is 

 {{'{'}}
      nzTop: '24px',
      nzRight: '0px',
      nzDuration: 4500,
      nzMaxStack: 7,
      nzPauseOnHover: true,
      nzAnimate: true
   {{'}'}}

API

Global Configuration (NZ_NOTIFICATION_CONFIG)

Parameters type Default Description
nzTop String 24px Top distance
nzRight String 0px Right distance
nzDuration Number 0 duration, does not disappear when set to 0 (available in NzNotificationService)
nzMaxStack Number 8 Prompt Maximum Stacks
nzPauseOnHover bool true Stop countdown when hovering (available in NzNotificationService)
nzAnimate bool true Switch Animation Effects (Available in NzNotificationService)

NzNotificationService Services

Tip: The following options parameter supports nzDuration / nzAnimate / nzPauseOnHover in the global configuration.
method Parameters Description
blank (title: string, content: string, options ?: Object) Tips without icons
success (title: string, content: string, options ?: Object) Success Prompt
error (title: string, content: string, options ?: Object) Failure Alert
warning (title: string, content: string, options ?: Object) Warning tip
info (title: string, content: string, options ?: Object) Message tip
create (type: string, title: string, content: string, options ?: Object) Provides a type attribute that passes in options such as 'success'
html (html: string, options ?: Object) Use html code to render content
remove (id ?: string) remove the specific id of the message, when id is empty, remove all messages
