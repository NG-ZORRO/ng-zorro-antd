Popover

Click / mouse to move the element to bring up the bubble card floating layer.

When to use #

  • When the target element has further description and related operation, it can be stored in the card and displayed according to the user's operation behavior.

    • The difference between

  • and Tooltip is that users can manipulate elements on the float so it can host more complex content such as links or buttons.

Code Demo

The simplest usage.

Mouse in, gather, click.

Location has twelve directions

Use the nzVisible property to control the float display

API

nz-popover

Parameters Description type Default
nzPlacement bubble box location, optional top left right bottom topLeft topRight
bottomLeft bottomRight leftTop leftBottom rightTop rightBottom 		String top
nzTemplate is used to define the content of the content ng-template
nzVisible is used to manually control the float explicit hidden bidirectional binding Boolean false
nzVisibleChange Show hidden callbacks EventEmitter None
nzTrigger Trigger behavior, optional hover / focus / click String hover
nzOverlayClassName card class name String None
nzOverlayStyle Card Style object None
nzMouseEnterDelay How long does it take for the mouse to move to show Tooltip in seconds (nzTrigger = 'hover') number 0
nzMouseLeaveDelay How long does it take for the mouse to hide Tooltip, seconds: (nzTrigger = 'hover') number 0.1
