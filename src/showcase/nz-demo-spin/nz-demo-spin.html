Spin 加载中

用于页面和区块的加载中状态。

何时使用 #

页面局部处于等待异步数据或正在渲染过程时，合适的加载动效会有效缓解用户的焦虑。

代码演示

一个简单的loading状态。

放入一个容器中。

自定义描述文案，指定的 nzTip 文案会直接代替 ...

小的用于文本加载，默认用于卡片容器级加载，大的用于页面级加载。

可以直接把内容内嵌到 nz-spin 中，将现有容器变为加载状态。

API

参数 说明 类型 默认值
nzSize spin组件中点的大小，可选值为 small default large Enum default
nzSpinning 用于内嵌其他组件的模式，可以关闭 loading 效果 Boolean true
nzTip 自定义描述文案 String 无
