Carousel 走马灯

旋转木马，一组轮播的区域。

何时使用 #

  • 当有一组平级的内容。

  • 当内容空间不足时，可以用走马灯的形式进行收纳，进行轮播展现。

  • 常用于一组图片或卡片轮播。

API

参数 说明 类型 默认值
