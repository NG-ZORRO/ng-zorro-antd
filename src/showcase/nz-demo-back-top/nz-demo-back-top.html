BackTop back to top

Returns the action button at the top of the page.

When to use #

  • When the page content area is longer;

  • When users need to frequently return to the top to view related content.

Code Demo

The simplest usage.

You can customize the style of the button back to the top, limiting the width and height: 40px * 40px .

Set the [nzTarget] parameter to allow the top of a container to be returned.

API #

nz-back-top #

has a default style, 50px , which can be overridden.

Custom style width is no greater than 40px * 40px.

Parameters Description type Default
nzTemplate Custom content, see example ng-template
nzVisibilityHeight The scroll height does not appear until nz-back-top number 400
nzClick Click the button's callback function -
nzTarget Sets the element that needs to listen for its scroll event with a value that returns a function of the corresponding DOM element Element window
