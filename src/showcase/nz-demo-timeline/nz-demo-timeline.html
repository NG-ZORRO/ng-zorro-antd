Timeline

Time-of-day stream information to be displayed vertically.

When to use

  • When there is a series of messages that need to be time-aligned from top to bottom;
  • Requires a timeline for visual concatenation;

Code Demo

Timeline basics usage.

Circle color, green for finished, success status, red for alarm or error status, blue for in progress or other default status.

Add a ghost node in the last position, said the timeline is not completed, still recording process. You can specify ng-template and #pending

You can set custom elements with ng-template and #custom .

API

nz-timeline-item

Parameters Description type Default
nzColor The color of the circle, fixed to "green", "red", "blue" 'green' | 'red' | 'blue' 'blue'
pending Ghost node element ng-template
custom Custom Elements ng-template
