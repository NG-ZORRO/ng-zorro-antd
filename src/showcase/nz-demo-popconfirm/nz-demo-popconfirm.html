Popconfirm

Click the element and a pop-up confirmation box will pop up.

When to use #

  • The operation of the target element needs to be further confirmed by the user. When the target element is displayed, a floating-layer prompt is popped up to ask the user.

  • The interactive form is lighter than the full-screen centered modal dialog box that confirm pops up.
