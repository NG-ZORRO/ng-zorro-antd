Menu

A menu list that provides navigation for pages and features.

When to use #

The navigation menu is the soul of a website, and users rely on navigation to jump through pages. Generally divided into top navigation and side navigation, the top navigation provides a global category and function, side navigation to provide multi-level structure to store and arrange the site architecture.

For more examples of layout and navigation, see: Generic Layout .

API

nz-menu

Parameters Description type Optional Default
nzTheme theme color string: light dark light
nzMode menu type, now supports vertical, horizontal, and embedded mode three string: vertical horizontal inline vertical
nzInlineCollapsed Controls the indentation / expansion of the embedded menu. Boolean false
nzClickActive Click to select the submenu Boolean true

nz-menu-submenu

Parameters Description type Optional Default
nzOpen Whether submenu is expanded or not. can be bound Boolean false
nzOpenChange submenu Expand Close Callback Func
[title] ng-content tag for submenu title content

nz-menu-group

Parameters Description type Optional Default
[title] ng-content tag, used to place menu-group title content

nz-menu-item

Parameters Description type Optional Default
nzSelected Is the current menu item selected Boolean false

nz-menu-item-divider

Divider line in between menu items, only used in vertical popup Menu or Dropdown Menu.
