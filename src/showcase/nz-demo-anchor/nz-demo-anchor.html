Anchor Anchor

is used to jump to the page specified location.

When to use #

You need to show the anchor links that are available on the current page for jumps and to jump quickly between anchors.

Code Demos

Used with nz-affix .

The simplest usage.

API #

nz-anchor #

after the specified offset has been reached from the top of the window
Parameters Description type Default
nzOffsetTop Trigger number
nzBounds Anchor Area Boundary number 5 (px)
nzScroll Trigger when scrolling to an anchor function

nz-link #

Parameters Description type Default
nzHref Anchor Link string
nzTitle text string
nzTemplate Text content overwrites nzTitle ng-template
