Tag Tag

Small tags for tagging and sorting.

When to use #

  • Attributes and dimensions used to mark things.

  • for classification.

Code Demo

The use of base tags can be turned off by adding nzClosable . Closes the tag with two events: nzBeforeClose nzClose .

Generate a set of labels with an array that can be dynamically added and deleted by listening to the nzClose event at the end of the animation.

Choose the topics you are interested in.

We added a variety of preset color label styles for use in different scenes. If the default value can not meet your needs, you can set to a specific color value.

Checkbox-like effects can be implemented with nz-checkable-tag and click to switch between the selected effects.

API

nz-tag

Parameters Description type Bidirectional binding Default
nzClosable Tags can be closed bool No false
nzColor tag color string No -
nzBeforeClose Close the previous event, the parameters are: (event: Event). You can prevent the tag from closing by setting event.preventDefault (); . Event No -
nzClose After closing the event, the argument is: (event: Event), the closed animation is finished. Event No -

nz-checkable-tag

Parameters Description type Bidirectional binding Default
nzChecked Set label's selected status bool No false
nzClosable Tags can be closed bool No false
nzColor tag color string No -
nzBeforeClose Close the event before the parameters: (Event: Event): void. You can prevent the tag from closing by setting event.preventDefault (); . Event No -
nzClose Close the event, the argument is: (event: Event): void, the closed animation is finished. Event No -
nzChange The event fired when the tag is clicked, with the parameters: (checked: boolean): void. Event No -
