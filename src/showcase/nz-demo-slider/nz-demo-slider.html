Slider Sliding Input Bar

A sliding input that shows current values ​​and selectable ranges.

When to use #

It can be continuous or discrete value when the user needs to choose between numeric range and custom range.

Code Demo

Basic slider. Renders as a double slider when range is true . When disabled is true , the slider is not available.

You can set icons to express business meaning around the slider.

Variables bound to ngModel are updated when the value of Slider changes (this can be monitored via setter or binding ngModelChange). The onAfterChange event is fired on onmouseup and the current value is passed in as a parameter.
