Cascader

Cascading Select Box.

When to use #

  • There is a need to choose from a set of associated data sets, such as provincial and municipal districts, company levels, and things categories.

  • When selecting from a large data set, use multi-level classification to facilitate selection.

  • Compared to the Select component, you can choose between the same float layer and have a better experience.
