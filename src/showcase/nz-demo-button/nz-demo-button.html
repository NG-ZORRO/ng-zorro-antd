Button

The button is used to start an instant action.

When to use

Marks an action command (or encapsulates a set of actions) in response to a user's click action to trigger the appropriate business logic.

Code Demo

There are four types of buttons: the main button, the second button, the dotted line button, the danger button. The main button appears in the same operating area at most once. where [nzType] = "'default'" can be reduced to nzType = "default", and will not be repeated here.

Buttons are large, medium and small size.

Set buttons to large and small sizes for large small by setting nzSize If you do not set nzSize, the size is medium.

Adding the loading attribute will cause the button to be loaded and the last two buttons will be clicked to enter the loaded state.

Multiple nz-button tags can be placed in the container of nz-button-group.

API

Generate different button styles by setting the button's properties, in the order of type - & gt; shape - & gt; size - & gt; loading - & gt; disabled

The properties of the

button are as follows:

attribute Description type Default
nzType Set the button type, optionally primary dashed danger default String -
nzShape Set button shape, optional circle, or none String -
nzSize Set button size, optional small large or no String default
nzGhost Set ghost button Boolean false
nzLoading Set button loading status Boolean false
