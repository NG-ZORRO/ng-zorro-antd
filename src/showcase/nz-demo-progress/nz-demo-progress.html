Progress Progress Bar

Shows the current progress of the operation.

When to use #

Shows the user the current progress and status of the operation when it takes a long time to complete.

  • When an operation interrupts the current interface or needs to run in the background and may take more than 2 seconds;

  • When you need to display the percentage of an operation completed.
