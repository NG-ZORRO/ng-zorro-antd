Alert Alert

Warnings to show the information you need to follow.

When to use

  • When a page needs to show the user a warning message.

  • Non-floating layer static display form, always show, does not disappear automatically, the user can click to close.

Show close button, click to close warning message.

Delicious icons make the type of information more visible.

A warning message with accompanying text introduction.
