DatePicker

Control to enter or select a date.

When to use #

When the user needs to enter a date, he can click on the standard input box to pop up the date panel to select.

Code Demo

The simplest usage.

Three sizes of input box, big used in the form, the default.

The selection box's status is not available.

You can set the nzDisabledDate method to constrain the start and end dates.

Use the nzFormat attribute to customize the date display format you need, such as YYYY / MM / DD.

Add the Select Time feature, whose properties are passed to the built-in nz-timepicker when nzShowTime is an object.

Set the nzDisabledDate method to determine a non-selectable period.

As in the example above, you can not select a date after today.

API

Parameters Description type Default
ngModel Impression Date Date current date
nzMode Selector mode, month only to the month, day to the day String "day"
nzSize Input box size, large height is 32px, small is 22px, default is 28px String month
nzFormat Date format for display, configuration reference Moment.js Docs String "YYYY-MM-DD"
nzDisabled Disable bool false
nzDisabledDate Callback function to disable date, true to disable this date function (date)
nzShowTime time option, see nz-timepicker parameter null
