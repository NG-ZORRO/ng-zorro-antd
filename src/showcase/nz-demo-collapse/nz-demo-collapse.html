Collapse 折叠面板

可以折叠/展开的内容区域。

何时使用

  • 对复杂区域进行分组和隐藏，保持页面的整洁。

  • 手风琴 是一种特殊的折叠面板，只允许单个内容区域展开。

API

nz-collapseset

参数 说明 类型 默认值
nzBordered 是否有边框 Boolean true
nzAccordion 是否是手风琴类型 Boolean false

nz-collapse

参数 说明 类型 默认值
nzTitle 面板头内容 String -
nzActive 当前tab是否被选中 Boolean false
nzDisabled 当前tab是否禁止选中 Boolean false
[collapse-title] 与nzTitle二选一，定义面板头内容 ng-content 无
