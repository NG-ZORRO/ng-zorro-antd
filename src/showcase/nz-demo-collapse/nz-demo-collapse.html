Collapse Panel

The content area that can be collapsed / expanded.

When to use

  • Group and hide complex areas to keep the page clean.

  • Accordion is a special collapsible panel that allows only a single content area to expand.

Code Demo

Multiple panels can be expanded simultaneously, this example by default expands the first one.

The accordion, open only one tab at a time. The first one is opened by default.

API

nz-collapseset

Parameters Description type Default
nzBordered is there a border Boolean true
nzAccordion Accordion type Boolean false

nz-collapse

Parameters Description type Default
nzTitle panel header content String -
nzActive The current tab is selected Boolean false
nzDisabled Whether to prohibit the current tab selected Boolean false
[collapse-title] Select one with nzTitle to define the header content ng-content None
