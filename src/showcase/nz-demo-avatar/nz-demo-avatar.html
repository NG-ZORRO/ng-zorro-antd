Avatar

is used to represent a user or thing, supporting pictures, icons, or characters.

Code Demos

Avatars come in three sizes, with two shapes available.

Three types are supported: images, icons, and characters, where Icon and character types customize the icon color and background color.

For character avatars, when the string is longer, the font size can be adjusted automatically based on the avatar width.

Normally used for message prompting.

API #

nz-avatar #
