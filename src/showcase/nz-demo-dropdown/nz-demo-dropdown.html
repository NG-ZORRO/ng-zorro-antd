Dropdown

Pop down the list.

When to use

Use this component to hold action elements when there are too many action commands on the page. Click or move contacts, a drop-down menu will appear. You can choose from the list and execute the corresponding command.

API

nz-dropdown

Note: You need to include the directive tag of [nz-dropdown] on the element that triggers the drop-down menu to locate the element's position. The attributes are as follows
Parameters Description type Default
[nz-dropdown] Element to locate the trigger drop-down menu Directive -
[nz-dropdown-custom] Custom pull-down menu content, not mix with the menu ng-content -
nzTrigger Triggered pulldown behavior 'click' | 'hover' 'hover'
nzClickHide Click to hide the menu Boolean true
nzVisible menu is displayed Boolean false
nzVisibleChange The menu is called when the state changes, the parameter is nzVisible Function (visible)
nzPlacement bottomLeft bottomCenter bottomRight topLeft topCenter topRight String bottomLeft

nz-dropdown-button

Parameters Description type Default
nzType button type String 'default'
nzSize button size String -
nzClick Click the left button's callback Function -
nzTrigger Triggered pulldown behavior 'click' | 'hover' 'hover'
nzClickHide Click to hide the menu Boolean false
nzVisible menu is displayed Boolean false
nzVisibleChange The menu is called when the state changes, the parameter is nzVisible Function -
nzPlacement bottomLeft bottomCenter bottomRight topLeft topCenter topRight String bottomLeft

The menu is available from nz-menu and includes menu items nz-menu-item nz-menu-divider and submenus nz-submenu
