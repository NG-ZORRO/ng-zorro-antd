Form

We provide the following three permutations for nz-form:

  • Horizontal arrangement: labels and form controls are arranged horizontally; (default)

  • Vertical arrangement: labels and form controls are arranged vertically;

  • Inline Arrangements: Form items are arranged horizontally within a row.

Form Fields

The form must contain the form fields. The form fields can be input controls, standard form fields, labels, pull-down menus, text fields, and so on.

[nz-form-control]

Called the form control area, usually used to place input, select and other components

Parameters Description type Default

nzValidateStatus The check status, which is automatically displayed asynchronously when it is defined as NgControl, can also be manually defined as string Optional: 'success' 'warning' 'error' 'validating' td> NgControl | string -
