Message

Global display operation feedback.

When to use #

  • Provides feedback on success, warnings, and errors.

  • The top center displays and disappears automatically, a lightweight reminder that does not interrupt user actions.

How to use

If you want to modify the default message configuration, you can modify the value of provider NZ_MESSAGE_CONFIG .

(For example, add '{{' {'}} in your module's provider:: NZ_MESSAGE_CONFIG, useValue: {{' {'}} nzDuration: 3000 {{'} '}} {{'}'}} , NZ_MESSAGE_CONFIG can be imported from ng-zorro-antd

The default is 

 {{ '{' }}
    nzDuration             : 1500,
    nzMaxStack             : 7,
    nzPauseOnHover         : true,
    nzAnimate              : true
 {{ '}' }}

API

Global Configuration (NZ_MESSAGE_CONFIG)

Parameters type Default Description
nzDuration Number 0 duration, does not disappear when set to 0 (available in NzNotificationService)
nzMaxStack Number 8 Maximum number of hints to show
nzPauseOnHover Boolean true Pause countdown when mouse over (available in NzNotificationService)
nzAnimate Boolean true Switch Animation Effects (Available in NzNotificationService)

NzMessageService Service

Tip: The following options parameter supports nzDuration / nzAnimate / nzPauseOnHover in the global configuration.
method Parameters Description
loading (content: string, options ?: Object) Loading
success (content: string, options ?: Object) Success Message
error (content: string, options ?: Object) failed message
warning (content: string, options ?: Object) Warning Message
info (content: string, options ?: Object) Information Message
create (type: string, content: string, options ?: Object) Provides a type attribute that passes in options such as 'success'
html (html: string, options ?: Object) Use html code to render content
remove (id ?: string) remove the specific id of the message, when id is empty, remove all messages
