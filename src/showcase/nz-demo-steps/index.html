Steps

Navigation bar that guides the user through the process.

When to use

  • When tasks are complex or there is a relationship, it is broken down into a series of steps to streamline tasks.

Code Demo

Simple steps

Mini version of the step bar, enabled by setting size="small".

You can enable custom icons by setting the icon attribute of nz-step.

Normally used in conjunction with content and buttons to indicate the progress of a process.

Simple vertical step bar.

A simple, vertical, small step bar.

Use the status attribute of Steps to specify the status of the current step.

Progress bar containing step points.

API

nz-steps

Parameters Description type Default
nzCurrent Specifies the current step, counting from 0. Within the child Step element, you can override the status via nzStatus Number 0
nzStatus Specifies the status of the current step, optionally wait process finish error String process
nzSize specifies the size and currently supports normal (default) and mini (small) String default
nzDirection Specifies the direction of the step bar. Currently there are two directions for horizontal and vertical vertical String horizontal
nzProgressDot Use the point style step by adding this attribute Boolean -

nz-step

Parameters Description type Default
nzStatus Specifies the status. When this property is not configured, the Steps current is used to automatically specify the state. Optional: wait process finish error String wait
nzTitle title String -
nzDescription Details of steps, optional. If you need to pass in TemplateRef, use template, see Example String or TemplateRef -
#nzIcon The type of step icon, optional ng-template -
