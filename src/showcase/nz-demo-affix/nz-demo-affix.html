Affix Fixing

Pin the page element in the visible area.

When to use #

When the content area is relatively long and the page needs to be scrolled, the operation or navigation corresponding to this part needs to be always displayed within the scrolling range. Commonly used for side menu and button combinations.

When the page visibility is too small, use this feature with caution to avoid obstructing the page content.
