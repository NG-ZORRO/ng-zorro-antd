TimePicker

Controls to enter or select a time.

When to use

When users need to enter a time, they can click on the standard input box, pop-up time panel to choose.

Code Demo

The simplest usage.

Three sizes of input box, big used in the form, the default.

Disable time selection.

Hide non-selectable options via nzHideDisabledOptions .

nzValue and nzValueChange need to be used together.

Do not show seconds, nor allow selection.

Limit the timeframe of 20:30 to 23:30 .
